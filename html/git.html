Git

  • git init – создаёт новый локальный репозиторий.
  • git clone [url] – клонирует удалённый репозиторий.
  • git add [файл] – добавляет файлы в индекс для следующего коммита.
  • git commit -m "сообщение" – фиксирует изменения с комментарием.
  • git push – отправляет изменения в удалённый репозиторий.
  • git pull – забирает изменения из удалённого репозитория.
  • git status – показывает состояние рабочих файлов.
